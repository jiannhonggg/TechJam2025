## Setup

### Change directory

```bash
cd .\\LynxWeb
```

### Install the dependencies:

```bash
npm install
```

Create the following file under LynxWeb:

".env.local"
```bash
VITE_OPENAI_API_KEY='openai_api_key_here'
```


## Get started

Start the dev server, and the app will be available at [http://localhost:3000](http://localhost:3000).

```bash
npm run dev
```
